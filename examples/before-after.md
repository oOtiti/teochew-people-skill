# Before / After

这些例子展示 `chaoshan-ren` skill 适合修正什么问题。

## 例子一：把潮州误写成全部潮汕

原文：

```text
潮汕文化就是潮州文化，潮汕人都喝功夫茶、看潮剧、拜老爷。
```

问题：

- 把潮州等同于全部潮汕。
- 使用“都”过度概括。
- “功夫茶”在潮汕语境中宜写作“工夫茶”。
- 把信俗写成所有人的共同实践。

改写：

```text
潮州是潮汕文化的重要源流和重镇，但潮汕文化圈还包括汕头、揭阳及海内外潮人社群。工夫茶、潮剧、英歌、粿品、侨批和各地年俗，共同构成了潮汕人关于家、乡、礼数与远方的记忆。不同地方、不同家庭的做法并不完全一样，真正值得写的，正是这些生活细节里的地方感。
```

## 例子二：把民俗写得猎奇

原文：

```text
营老爷是潮汕最神秘的迷信活动，全村人都会疯狂参加。
```

问题：

- “神秘”“迷信”“疯狂”带有猎奇和贬损色彩。
- 把地方性信俗写成全潮汕统一现象。

改写：

```text
在潮汕部分地区，营老爷是春节前后或地方节庆中很有代表性的社区活动。锣鼓、标旗、英歌、潮剧和巡游队伍把村社的人重新聚到一起，也把一地的信俗、秩序和年味呈现出来。各地时间、路线和仪式不同，写它时更适合用尊重而中性的语言，而不是把它当成猎奇景观。
```

## 例子三：菜品文案太泛

原文：

```text
潮汕牛肉火锅，正宗古法，潮汕人都爱吃。
```

问题：

- “正宗古法”缺少依据。
- “潮汕人都爱吃”过度概括。
- 文案没有写出风味重点。

改写：

```text
潮汕牛肉火锅好吃，靠的是新鲜、刀工和时间。不同部位切得薄厚有别，入锅只需片刻，再蘸一点沙茶或普宁豆酱，牛肉的鲜甜就出来了。它是很多人认识潮汕味道的入口，但潮汕饮食远不止这一锅热气。
```
