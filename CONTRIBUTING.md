# 贡献指南

欢迎补充潮汕文化资料、写作模板、来源链接和任务示例。这个项目最看重两件事：准确和有分寸。

## 可以贡献什么

- 增加某个地方更具体的文化说明，例如普宁英歌、潮阳英歌、揭阳行彩桥、潮汕粿品、侨批故事。
- 补充可靠资料来源，优先政府、方志、博物馆、非遗、学术或媒体深度报道。
- 增加更好的写作模板、短视频脚本示例、审校案例和反面示例。
- 修正过度概括、潮州/潮汕范围混淆、刻板印象或不准确说法。

## 不建议贡献什么

- 没有来源的“古法”“祖训”“禁忌”“潮汕话读音”。
- 把某个村、某个姓氏、某个家庭的做法写成全部潮汕通用。
- 猎奇化宗族、信俗、女性、婚嫁或地方神明的内容。
- 大段复制受版权保护的文章。

## 写作约定

- `skills/chaoshan-ren/SKILL.md` 保持轻量，只放触发、边界、流程和导航。
- 详细内容放到 `skills/chaoshan-ren/references/`，按主题拆分。
- 新增事实性内容时，尽量同步更新 `90-资料来源.md`。
- 涉及地方差异时，用“部分地区”“不少家庭”“常见于”这类表达。

## 提交前验证

```powershell
npm run validate
npm run pack:check
```

如果启用了 Git hook，提交时也会自动运行 skill 验证。
