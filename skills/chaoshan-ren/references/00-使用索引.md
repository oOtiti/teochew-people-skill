# 使用索引

这份索引用于快速决定读取哪份参考资料。不要一次性加载所有文件；根据用户任务选择最相关的一两份。

## 任务到资料

- 用户问“潮汕包括哪里”“潮州和潮汕什么关系”“Teochew 是什么”：读 `01-范围与称谓.md`。
- 用户问出花园、婚嫁、待客、祭祖、拜老爷、家庭礼数：读 `02-风俗礼仪.md`。
- 用户问春节、元宵、七样羹、行彩桥、营老爷、清明、冬至：读 `03-节庆岁时.md`。
- 用户问工夫茶、潮汕菜、粿、牛肉火锅、小吃、菜单文案：读 `04-饮食与工夫茶.md`。
- 用户问潮汕话、潮剧、英歌、工艺、祠堂建筑、非遗：读 `05-语言戏曲英歌工艺.md`。
- 用户问侨乡、侨批、海外潮人、善堂、宗亲、当代变化：读 `06-侨乡与社会组织.md`。
- 用户要成稿、改写、标题、短视频脚本、祝福语、审校意见：读 `07-写作模板与审校清单.md`。
- 用户需要出处、引用、核对事实：读 `90-资料来源.md`。

## 最小读取建议

- 普通文案：`07-写作模板与审校清单.md` + 对应主题文件。
- 文化解释：`01-范围与称谓.md` + 对应主题文件。
- 涉及非遗或事实判断：对应主题文件 + `90-资料来源.md`。
- 涉及仪式步骤：对应主题文件即可；不要把资料写成统一操作手册。
