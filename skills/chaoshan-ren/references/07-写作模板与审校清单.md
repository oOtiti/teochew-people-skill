# 写作模板与审校清单

## 文化解释模板

```text
先写一个具体场景。
再说明这个习俗/物件/表演是什么。
接着解释它在潮汕生活中的意义。
然后提醒地方和家庭差异。
最后落到今天的人情、记忆或传承。
```

示例骨架：

```text
在潮汕，很多话不是站着说完的，而是坐到茶盘边慢慢说开的。工夫茶不只是泡茶方法，也是待客和相处的节奏。客人来了，先烧水、温杯、斟茶，几杯小茶之间，长辈的问候、亲友的近况、邻里的情分也一起续上了。当然，各地各家的茶习惯不同，重要的不是固定动作，而是那份把人请进来、坐下来、慢慢谈的心意。
```

## 短视频脚本模板

```text
开头：一个具体画面或反差。
中段：解释文化元素，避免百科腔。
转折：说明不是全潮汕统一，点出地方差异。
结尾：回到亲情、乡音、节令或当代生活。
```

## 展览/文旅文案模板

```text
潮汕文化不是单一城市的标签，而是粤东海陆之间长期形成的生活共同体。工夫茶、潮剧、英歌、粿品、侨批、祠堂与善堂，共同构成了潮汕人关于家、乡、礼、艺和远方的记忆。
```

## 家族祝福模板

```text
今天借着这个日子，一家人坐在一起，先谢长辈的照顾，也祝孩子从此更懂事、更有担当。潮汕礼俗讲究的是心意和传承，具体做法各家不同，但共同的愿望都是平安、长进、惜福、念亲。
```

## 菜品介绍模板

```text
这道菜的好，不在重味，而在分寸。潮汕菜常讲究鲜、清、准，火候不能过，蘸料也不能抢。入口看似简单，背后是对食材和手艺的尊重。
```

## 审校清单

检查一段潮汕文化内容时，优先看：

- 是否把潮州等同于全部潮汕。
- 是否忽略汕头、揭阳、普宁、潮阳、潮南等地的存在。
- 是否使用“潮汕人都”“必须”“一定”等过度概括。
- 是否把家庭或村落做法写成统一民俗。
- 是否猎奇化信俗、宗族、性别议题或民间表演。
- 是否编造潮汕话读音、祖训、禁忌、古法。
- 是否把牛肉火锅、英歌、工夫茶当成潮汕文化全部。
- 是否需要资料来源支撑“非遗”“起源”“历史年代”“官方称号”等判断。

## 常用改写方向

- 从“潮汕人都很会做生意”改为“潮汕侨乡和商业网络确实塑造了许多跨地域经验，但潮汕人的生活远不止商业。”
- 从“潮州就是潮汕”改为“潮州是潮汕文化的重要源流和重镇，但潮汕文化圈还包括汕头、揭阳等地及海外潮人社群。”
- 从“拜老爷很神秘”改为“营老爷等地方信俗常和村社节庆、祈愿、锣鼓、英歌、潮剧一起出现，是社区记忆的一部分。”
